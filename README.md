## Nuke.js
How use:
```js
const Nuke = require('nuke.js');
new Nuke('Nuclear WarHead', {
 interval: 5000,
 timeout: 3000,
});
```
- class Nuke
- Nuke.type (string)
>The type of your nuke. Default atom bomb
- Nuke.options (object)
- options.interval
>Interval in milliseconds for your Nuke to explode.
- options.timeout
>Timeout in milliseconds for your Nuke to explode.
